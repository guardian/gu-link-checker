Guardian link checking

Checking what the Guardian links to

What is this?

This application uses the Content API to examine what links are present in the body and then tries to validate them.

Current rules

  • For a regular link the app will try and read the link
  • Links are assumed to absolute and will fail if they are relative
  • For a mailto link the app will try to check that the content of the link is a valid email
  • For sponsored content the app will check whether the link has had the rel="nofollow" attribute applied to it
